使用 expect/git hooks 实现项目在服务器端的自动部署

10月 28, 2016 in Code

项目在本地被开发，之后肯定要被送到服务器上。每次都用 scp/rsync 送上去随后重跑服务未免过于繁琐；特别是在公司内网还存在着堡垒机的情况下，如果要把文件送上目标机器，必须要通过堡垒机做中转，每次手打长串命令很麻烦。使用 expect/git hooks可以简化流程，实现一行指令让项目自动部署在服务器上。
